Exercise (Set of finite-length tuples)

Define using mathematical set notation, the set of all finite-length tuples, where each element of the tuple is an element of \(\mathbb Z\).

Solution

Here is one way of defining this set using the set-builder notation: \[\{(n_1, n_2, \ldots, n_k) : k \in \mathbb N\text{ and } \forall i \in \{1, 2, \ldots, k\}, n_i \in \mathbb Z\}.\] The way we would read this out loud is: “The set of all tuples with elements \(n_1\), \(n_2\), to \(n_k\) such that \(k\) is a natural number, and for all \(i\) from \(1\) to \(k\), \(n_i\) is in \(\mathbb Z\).” Note that \(k\) here is not some fixed number; we consider all possible values of \(k\). Therefore this set includes all possible finite-length tuples.

We can also use an alternative notation. For a set \(A\), it is common to use \(A^k\) to denote the Cartesian product of \(A\) with itself \(k\) times, i.e., \[A^k = \underbrace{A \times A \times \cdots \times A}_{k \text{ times}}.\] Equivalently, \[A^k = \{(a_1, a_2, \ldots, a_k) : \forall i \in \{1, 2, \ldots, k\}, a_i \in A\}.\] (Note that in this case, \(k\) is some fixed value, so \(A^k\) consists only of tuples of the same length, \(k\).) With this notation, the set we want to define for this exercise can be written as the following union: \[\mathbb Z^0 \cup \mathbb Z^1 \cup \mathbb Z^2 \cup \mathbb Z^3 \cup \ldots\] Or more simply: \[\bigcup_{k \in \mathbb N} \mathbb Z^k.\] Note that \(A^0\) consists of the empty tuple \(()\). And in this course, \(\mathbb N\) contains \(0\).

Exercise (Defining functions)

What is the typical way of defining a function \(f: A \to B\)? When do you need to argue that a function that you defined is well-defined?

Solution

When we want to define a function from a set \(A\) to a set \(B\), we need to specify, for all \(a \in A\), what \(f(a) \in B\) is, and the value of \(f(a)\) must be unambiguous. Sometimes this can be done in a single shot, where you consider an arbitrary \(a \in A\) and define what \(f(a)\) is. For example, the increment function \(f : \mathbb N\to \mathbb N\) can be defined as follows: for all \(n \in \mathbb N\), \(f(n) = n+1\). Sometimes we need to consider different cases for the input and make our definition accordingly. For example, we could define a parity function \(f: \mathbb N\to \mathbb N\) as follows. For all \(n \in N\), let \[f(n) = \begin{cases} 0 & \text{ if $n$ is even,} \\ 1 & \text{ if $n$ is odd.} \end{cases}\] Sometimes we may need to specify the output for each individual input separately. For example, we could define a function \(f : \{1,2,3,4\} \to \{0,1\}\) as follows: \[f(1) = 0, \quad f(2) = 1, \quad f(3) = 1, \quad f(4) = 0.\]

In all of the above cases, we unambiguously specify, for each possible input in \(A,\) what the corresponding output in \(B\) is. Once we do this, the function is well-defined, and there is nothing more that needs to be done.

That being said, in certain special situations, an argument may be needed to show that the function you are defining is well-defined. Consider, for example, a function \(f : \mathbb Q\to \mathbb Z\) defined as follows. For any \(a/b \in \mathbb Q,\) let \(f(a/b) = a+b.\) In this case, there is actually ambiguity, because a rational number (an element of \(\mathbb Q\)) could be represented as a fraction \(a/b\) in multiple different ways. For a function to be a function, each input must get mapped to single element in the range. But this is not the case for the \(f\) we defined. The fractions \(a/b\) and \(2a/2b\) represent the same rational number, but \(f(a/b)\) is not necessarily the same as \(f(2a/2b).\) And therefore \(f\) is not a well-defined function.

So in some situations, we need to be careful that the function we define is indeed well-defined. We need to be aware of this. But in many situations, when we define a function, we unambiguously specify the output for each possible input, and that is sufficient.

If you would like more detail on this, see this blog post by Timothy Gowers.

Exercise (Types of functions)

What is an injective function, a surjective function, and a bijective function?

Solution

A function \(f : A \to B\) is injective if for all \(a\) and \(a'\) in \(A\) such that \(a \neq a'\), we have \(f(a) \neq f(a')\). So \(f\) does not map two different elements of \(A\) to the same element in \(B\). One can think of an injective function from \(A\) to \(B\) as a labeling mechanism. We think of \(A\) as some set of objects we want to label, and we think of \(B\) as the possible labels. The injective function \(f\) specifies a unique label for each element of \(A\). So given some label \(b \in B\), there is at most one \(a \in A\) that is assigned that label.

A function \(g : A \to B\) is surjective if for every \(b \in B\), there exists \(a \in A\) such that \(f(a) = b\). In words, for every element in \(B\), there is an element in \(A\) that maps to it, so in this sense, you might say \(B\) is “covered” (under \(g\)). Note that two different elements of \(A\) could map to the same element in \(B\), so \(g\) need not be injective. However, if a function is both injective and surjective, then we call it bijective, and in this case, there is a one-to-one correspondence between the elements of \(A\) and the elements of \(B\).

Exercise (Showing injectivity)

Suppose you are asked to show that there is an injection from a set \(A\) to a set \(B\). What are the steps for doing this?

Solution

The typical way of showing the existence of an injection from set \(A\) to set \(B\) is to first define a function \(f : A \to B\), and then prove that \(f\) is indeed injective. We have already talked about how to define a function and whether you need to argue that the function is well-defined. To show that a function is injective, you can directly use the definition and show that for arbitrary \(a, a' \in A\) such that \(a \neq a'\), we have \(f(a) \neq f(a')\). Alternatively, you can argue the contrapositive: if \(f(a) = f(a')\), then \(a = a'\). Consider both options since one may end up being more convenient than the other.

It is a well-known fact that there is an injection from \(A\) to \(B\) if and only if there is a surjection from \(B\) to \(A\). So another way to show that there is an injection from \(A\) to \(B\) is to show that there is a surjection from \(B\) to \(A\). For this, you would first define a function \(g : B \to A\), and then argue that it is a surjection, i.e., argue that for all \(a \in A\), there is a \(b \in B\) such that \(g(b) = a\).

Exercise (Characteristic function)

Let \(X\) be some set. Recall that \(\wp(X)\) denotes the power set of \(X\), so \(\wp(X)\) is the set containing all the subsets of \(X\). Let \(\mathbf{F}(X)\) denote the set containing all the functions of the form \(f : X \to \{0,1\}\). Show that there is a one-to-one correspondence between \(\wp(X)\) and \(\mathbf{F}(X)\), i.e. there is a bijection between them. You don’t have to write a formal proof here, but given \(S \in \wp(X)\), think about the function \(f_S \in \mathbf{F}(X)\) that naturally corresponds to \(S\).

Solution

For any \(S \in \wp(X)\), i.e. for any \(S \subseteq X\), we can define the corresponding function \(f_S : X \to \{0,1\}\) as follows. For all \(x \in X\), let \[f_S(x) = \begin{cases} 1 & \text{ if $x \in S$,} \\ 0 & \text{ otherwise.} \end{cases}\] In the literature, \(f_S\) is called the characteristic function of \(S\). This terminology is very commonly used, so we encourage you to remember it.

Similarly if you are given any function \(f : X \to \{0,1\}\), the corresponding set \(S \in \wp(X)\) is defined so that \(x \in S\) if and only if \(f(x) = 1\).
