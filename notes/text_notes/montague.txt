These notes provide an overview of Montague Grammar, a formal framework for natural language semantics that treats English as a formal language using intensional logic and lambda calculus.
I. Foundations of Montague Grammar
Richard Montague’s seminal works (1970–1973) established that there is no theoretical difference between the syntax/semantics of "natural" languages (like English) and "artificial" languages (like logic).
Core Semantic Types
• t (Terms): Roughly corresponding to entities/individuals.
• f (Formulae): Truth values (True/False).

II. Semantic Rules and Lambda Expressions
The system uses \lambda-abstraction to build complex meanings.
Determiners (Quantifiers)
• "Some" / "A": \lambda P. \lambda Q. \exists x (P(x) \wedge Q(x))
• "Every": \lambda P. \lambda Q. \forall x (P(x) \to Q(x))
• "No": \lambda P. \lambda Q. \forall x (P(x) \to \neg Q(x))
Rules of Composition
• S \to NP + VP: The NP (quantifier) consumes the VP (predicate).
• NP \to Name: Type-lifted to \lambda P. P(name) to match the quantifier type.
• VP \to TV + NP: \lambda x. (NP(\lambda y. TV(y)(x)))
III. Key Semantic Problems & Solutions
1. The Unicorn Problem (Intensionality)
How do we distinguish between "John finds a unicorn" (implies a unicorn exists) and "John seeks a unicorn" (does not imply existence)?
• Meaning Postulates: For "extensional" verbs like find, we use a postulate to reduce the abstract relation to a concrete individual.
• Intensional Verbs: Verbs like seek remain at the level of properties/intensities, avoiding the existence commitment.
2. Scope Ambiguity
Sentence: "Every man loves a woman."
• Weak Scope (\forall \exists): Every man has some woman he loves (different women).
• Wide Scope (\exists \forall): There is one specific woman whom every man loves.
• Solution: Montague uses Derivational History (quantifying in) to produce two different logical formulas from one surface structure.
3. Intensional Logic & Possible Worlds
• Extension: The actual thing/truth value in a specific world.
• Intension: A function that takes a possible world and returns an extension (the "concept" across all possibilities).
• The Temperature Paradox: "The temperature is 90" and "The temperature is rising." If we only used extensions, we couldn't explain why "90 is rising" is a bad inference. "Rising" applies to the intension (the function over time), not the current value.
IV. Advanced Topics
Negative Polarity Items (NPIs)
Words like "yet" or "ever" must be "licensed" by negation or downward-entailing contexts.
• Correct: "He hasn't arrived yet." (Licensed by 'not')
• Incorrect: "He has arrived yet."
• Downward Entailment: An NPI is valid if the context allows inference from a set to a subset (e.g., "No man walks" \to "No father walks").
Paradoxes in Semantics
• Liars Paradox:"This sentence is false." In Montague's system, this can't be built because the sentence's self-reference doesn't exist in the formal hierarchy.
• Synonymy Paradox: If "groundhog" and "woodchuck" refer to the same animal (extension), then "I believe Phil is a groundhog" should imply "I believe Phil is a woodchuck." However, if the speaker doesn't know they are the same, the truth value differs. This requires Intensional Structure rather than just truth values.
V. Pragmatics
Meaning is a function taking a Possible World + a Context.
• Sense/Intension: Handled by the world.
• Context: Handles speaker-dependent terms (indexicals) like "I" or "here."
• Example: "I am hungry" depends on who is speaking at what time, even if the world stays the same.
